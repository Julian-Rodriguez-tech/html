Guia: como crear una pagina web con HTML

Introduccion

En esta guia aprenderas a crear una pagina web html desde 0




1. Crea el archivo .html

-- Para comenzar, en tu editor, en este caso Visual Studio Code , abre la carpeta donde deseas guardar tu trabajo.

-- Luego crea un archivo en la carpeta y dale el nombre **index.html**



2. Estructura basica

Una vez tengas creado tu archivo, sea manualmente o utilizando atajos, escribe la estructura básica del documento HTML. Puedes usar la siguiente si lo deseas: 

                
                <!DOCTYPE html>
                <html lang="es">
                   <head>
                      <meta charset="UTF-8" />
                      <meta name="viewport" content="width=device-width, initial-scale=1.0" />
                      <title>Mi Primer Sitio Web</title>
                   </head>
                   <body>
                      <!-- Contenido de la página va aquí-->
                   </body>
                </html>


3. Agregar contenido

A continuación, puedes agregar diversos elementos dentro del <body> para construir tu página:

  • <p> para párrafos
  • <h1> a <h6> para encabezados
  • <ul> y <ol> para listas
  • <img> para imágenes
  • <a> para enlaces
  • presiona aqui para ver mas etiquetas

4.Ejemplo

Aqui hay una breve desmostracion de una pagina web muy basica. 

                    
                     <!DOCTYPE html>
                     <html lang="es">
                  <head>

                     <meta charset="UTF-8" />
                     <meta name="viewport" content="width=device-width, initial-scale=1.0" />
 
                     <title>Mi Primer Sitio Web</title>
                  </head>
          <body>
             <header>
                     <h1>titulo de ejemplo</h1>
                     <p>ejemplo ejemplo ejemplo</p>

             </header>

            <hr>      
             <main>

               <section>
 
                     <h2>1. ejemplo</h2>
                             
                            
                     <ol>1 ejemplo </ol>
                             
                     <ol>2 ejemplo </ol>

                     <ol>3 ejemplo </ol>

               </section>    
  
             </main>
    
         </body>
                     </html>

5. Subir tu Trabajo a GitHub

Sigue estos pasos para subir tu archivo index.html a GitHub :

  1. Abre la terminal y navega a la carpeta de tu proyecto.
  2. Inicializa un repositorio de Git: git init
  3. Añade todos los archivos al repositorio: git add .
  4. Realiza el primer commit: git commit -m "First commit: HTML file"
  5. Crea un nuevo repositorio en GitHub.
  6. Conecta tu repositorio local con el remoto y sube los cambios:
    7. 
git remote add origin https://github.com/tu_usuario/tu_repositorio.git
git branch -M main
git push -u origin main
